# Israel Dafna - Landing Page

A client project: a Hebrew-language landing page built for a personal coaching and spiritual guidance service. The goal was a warm, trustworthy design that clearly communicates the offer and makes it easy for visitors to get in touch.

Live site: https://sarale-gamliel.github.io/israeldafna/

## Features

A hero section with a clear value proposition, a section addressing common pain points for the target audience, an explanation of the approach and process, and a contact section with a form plus direct phone and email links, including a clipboard fallback for copying the email address.

## Tech stack

Plain HTML, CSS, and JavaScript, deployed with GitHub Pages.

## Getting started

This is a static site. Open index.html directly in a browser, or serve the folder with any static file server.
